Source Reactor

This is a reactor named "Source" that emits a sequence of integers.

Parameters

  • sequence - An array of integers representing the sequence to emit (e.g., [0, 1, 2]).
  • n_sequence - The number of values to emit from the sequence (e.g., 3).

Output Ports

  • out - The output port for emitting integers.

State Variables

  • count - The state variable that keeps track of the current position in the sequence.

Logical Actions

  • next - The logical action used to control the emission of values.

Reactions

This reaction is triggered by the "startup" event and emits values from the sequence.

Parameters

  • startup - The event that triggers this reaction.
  • next - The logical action that controls the emission.

Return Value

The emitted value ("out") and the next logical action ("next").

Action Steps

  1. Set the "out" port to the value from the sequence at the current position.
  2. Increment the "count" state variable to move to the next value in the sequence.
  3. Check if there are more values to emit. If so, schedule the "next" action.
